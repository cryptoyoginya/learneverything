# <Эмодзи> <Сфера> — учебный курс learneverything

Это vault курса, который ведёт скилл `learneverything` (`~/.claude/skills/learneverything/`). Продолжение курса, разбор ответов, следующий урок — всё через него: прочитай `SKILL.md` скилла и глобальный журнал `~/Documents/learning/learning-log.jsonl`, затем действуй по пайплайну.

## Что где лежит

| Файл | Что это |
|---|---|
| `Профиль ученика.md` | цель, срок, уровень, предпочтения — источник правды о пользователе |
| `Карта темы.md` | полное дерево подтем из эталонов покрытия, пометки `[цель]` |
| `Источники/` | согласованный список: `01 Научная база` … `04 Видео и курсы` (нумерация = пирамида доказательности) + `Эталоны покрытия.md` |
| `План обучения.md` | модули и чекбоксы «смогу…», прогресс |
| `Уроки/` | уроки по порядку; пользователь отвечает в конце и ставит `???[вопрос]` в тексте |
| `Понятия/`, `MOC.md`, `Терминология.md`, `Практические выводы.md` | база знаний, пополняется после модулей |

## Правила

- Ответы и пометки `???` пользователя в уроках — данные, не мусор: не удалять и не переформатировать.
- Уроки задним числом не редактируются; правки смысла — только новым уроком или заметкой.
- Журнал `learning-log.jsonl` — только дописывание.
- Стиль любых новых текстов — `references/style-rules.md` скилла.

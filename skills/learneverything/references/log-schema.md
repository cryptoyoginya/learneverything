# Схема learning-log.jsonl

Файл: `~/Documents/learning/learning-log.jsonl`. Глобальный для всех сфер. Только дописывание строк, никогда не редактировать и не удалять существующие.

Каждая строка — одно событие:

```json
{"date": "YYYY-MM-DD", "sphere": "Имя сферы (= имя папки vault)", "event": "<тип>", ...поля типа}
```

## Типы событий

**`lesson_done`** — урок прочитан и разобран:
```json
{"date": "2026-07-23", "sphere": "Имя сферы", "event": "lesson_done", "lesson": "03 Название урока", "concepts": ["Понятие 1", "Понятие 2", "Понятие 3"], "questions_total": 4, "questions_ok": 3, "user_flags": ["не поняла Понятие 2"]}
```
`user_flags` — суть `???`-пометок и вопросов пользователя своими словами.

**`quiz_gate`** — итог квиз-гейта:
```json
{"date": "...", "sphere": "...", "event": "quiz_gate", "lesson": "...", "passed": true, "weak_concepts": []}
```

**`gate_skip`** — обход квиз-гейта («хочу дальше всё равно»):
```json
{"date": "...", "sphere": "...", "event": "gate_skip", "lesson": "...", "unresolved": ["понятие 1", "понятие 2"]}
```
Понятия из `unresolved` возвращаются в расписание повторений с повышенным приоритетом, пока не наберут 3 успешных извлечения.

**`retrieval`** — результат повторения по расписанию:
```json
{"date": "...", "sphere": "...", "event": "retrieval", "concept": "Понятие 1", "ok": true, "streak": 2}
```
`streak` — счётчик успешных извлечений в разных сессиях. `streak >= 3` — понятие усвоено (successive relearning), из расписания снимается. Неудача сбрасывает streak в 0.

**`recommendation_viewed`** — пользователь смотрел/читал что-то из «Куда дальше»:
```json
{"date": "...", "sphere": "...", "event": "recommendation_viewed", "item": "лекция из «Куда дальше»", "user_note": "что зацепило, своими словами"}
```
Следующий урок вплетает это в сократовский разбор: «ты смотрела лекцию про X — что зацепило?»

**`graded`** — оценка и фидбэк за урок (оценка только если включена в Профиле; фидбэк даётся всегда, в журнал идёт его суть):
```json
{"date": "...", "sphere": "...", "event": "graded", "lesson": "01 Название", "grade": 8, "note": "суть фидбэка: что удалось, что подтянуть (feedforward)"}
```
`grade` — 1–10 или `null`, если оценки выключены. По `grade` видно динамику от урока к уроку.

**`module_done`** — модуль закрыт, заметки написаны:
```json
{"date": "...", "sphere": "...", "event": "module_done", "module": "Модуль 2. Название модуля", "notes_created": 6}
```

**`course_event`** — крупные вехи: курс создан, план согласован, источники согласованы, курс завершён:
```json
{"date": "...", "sphere": "...", "event": "course_event", "kind": "created|sources_approved|plan_approved|finished", "note": "..."}
```

## Как читать при продолжении курса

1. Отфильтруй строки нужной сферы.
2. Последний `lesson_done` — где остановились; его `user_flags` — что разобрать.
3. Все `retrieval` со `streak < 3` + все `unresolved` из `gate_skip` — кандидаты в повторение следующего урока (приоритет: gate_skip, потом давность).
4. `recommendation_viewed` без последующего разбора — вплести в диалог.
